This tool converts the tweet.js file in your twitter archive into a json file which can be imported into roam. It is a static application, so no information you enter or file you upload here ever leaves your computer. Trust me, I don't want to know who you are so much more than you dno't want me to know who you are. Or don't trust me, hit ctrl-shift-I (assuming you're using chrome, otherwise google "whatever-browser developer tools"), open the "network" tab, upload a test file and watch nothing get sent.

Note: This tool is experimental and fairly untested, use at your own risk. If you get really mad you can shout at me @ollybot_redux. I would recommend spinning up a new graph to test it once before you integrate it into your existing setup.

I also expect to improve this in the future (through the use of block refs for a start), and roam does not support updating blocks via import, so you might want to be willing to delete whatever graph you've got the tweets loaded into when/if a new version of this rolls around.
